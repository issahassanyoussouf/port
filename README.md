# Portfolio trilingue ultra professionnel

Version moderne et responsive du portfolio d'Issa Hassan Youssouf.

## Langues incluses
- Français
- English
- العربية

## Fonctionnalités
- Site responsive mobile, tablette et ordinateur
- Changement de langue instantané FR / EN / AR
- Support RTL pour l'arabe
- Mode clair / sombre
- Projets filtrables et recherche instantanée
- Fenêtre de détail pour chaque projet
- Formulaire de contact avec validation et préparation d'un email
- SEO de base et balises Open Graph
- Images organisées dans `assets/img`

## Utilisation
Ouvrir `index.html` dans un navigateur moderne.

## Déploiement GitHub Pages
1. Envoyer tout le contenu de ce dossier dans un dépôt GitHub.
2. Aller dans Settings > Pages.
3. Choisir la branche principale et le dossier racine.
4. Publier.
